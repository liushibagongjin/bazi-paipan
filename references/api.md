# 八字排盘计算器 API 使用说明

## 概述
这是一个纯 Python 实现的八字排盘计算器，无第三方依赖，支持命令行和模块调用两种方式。

## 环境要求
- Python 3.7+（推荐 3.14）
- 无需安装任何第三方库

## 方法一：命令行调用

### 基本语法
```bash
python bazi_calculator.py --name <姓名> --time "<出生时间>" --location <出生地> --gender <性别> --liunian <流年年份> --output <输出文件>
```

### 参数说明
| 参数 | 必填 | 说明 | 默认值 | 示例 |
|------|------|------|--------|------|
| `--name` | 否 | 姓名 | 空字符串 | `--name 张三` |
| `--time` | 是 | 出生时间 | - | `--time "1990-05-15 08:30"` |
| `--location` | 否 | 出生地 | 北京 | `--location 成都` |
| `--gender` | 否 | 性别 | 男 | `--gender 女` |
| `--liunian` | 否 | 流年年份（格式：YYYY） | 当前年份 | `--liunian 2025` |
| `--output` | 否 | 输出文件路径 | 控制台 | `--output result.json` |

### 使用示例

#### 示例1：输出到控制台
```bash
python bazi_calculator.py --name 张三 --time "1990-05-15 08:30" --location 成都 --gender 男
```

#### 示例2：输出到指定文件
```bash
python bazi_calculator.py --name 张三 --time "1990-05-15 08:30" --location 成都 --gender 男 --output result.json
```

#### 示例3：输出到指定目录（自动生成文件名）
```bash
python bazi_calculator.py --name 张三 --time "1990-05-15 08:30" --location 成都 --gender 男 --output "D:\WorkBuddy Date\八字排盘"
# 自动保存为：D:\WorkBuddy Date\八字排盘\bazi_result_20260326_131730.json
```

#### 示例4：最简调用（使用默认值）
```bash
python bazi_calculator.py --time "1990-05-15 08:30"
# 默认：北京，男，无姓名，流年为当前年份，输出到控制台
```

#### 示例5：指定流年
```bash
python bazi_calculator.py --time "1990-05-15 08:30" --location 成都 --gender 男 --liunian 2025
# 计算2025年流年的八字分析
```

#### 示例6：查看未来流年
```bash
python bazi_calculator.py --time "1990-05-15 08:30" --location 成都 --gender 男 --liunian 2030
# 计算2030年流年的八字分析
```

## 方法二：Python 模块调用

### 导入函数
```python
from bazi_calculator import calculate_bazi, print_bazi
```

### 函数签名
```python
def calculate_bazi(
    birth_time_str: str,  # 出生时间，格式：YYYY-MM-DD HH:MM
    location: str,        # 出生地（城市名或经度,纬度）
    gender: str,          # 性别：'男' 或 '女'
    name: str = '',       # 姓名（可选，默认为空）
    liunian: str = None,  # 流年年份（可选，格式：YYYY，默认为当前年份）
    timezone_offset: float = 8.0  # 时区偏移（小时），默认北京时间
) -> Dict:  # 返回字典格式的八字排盘结果
```

### 函数签名（print_bazi）
```python
def print_bazi(
    result: Dict,         # calculate_bazi 返回的结果
    output_file: str = None  # 输出文件路径（可选，留空则输出到控制台）
) -> None
```

### 使用示例

#### 示例1：基本计算
```python
from bazi_calculator import calculate_bazi, print_bazi

result = calculate_bazi(
    birth_time_str="1990-05-15 08:30",
    location="成都",
    gender="男",
    name="张三"
)

# 输出到控制台
print_bazi(result)

# 或者输出到文件
print_bazi(result, "result.json")
```

#### 示例2：使用经纬度
```python
result = calculate_bazi(
    birth_time_str="1990-05-15 08:30",
    location="104.07,30.67",  # 成都的经纬度
    gender="男",
    name="张三"
)
```

#### 示例3：访问结果数据
```python
result = calculate_bazi("1990-05-15 08:30", "成都", "男", "张三")

# 读取基本信息
name = result['基本信息']['姓名']
birth_time = result['基本信息']['出生时间（公历）']
gender = result['基本信息']['性别']

# 读取四柱
year_pillar = result['四柱']['年柱']['干支']
month_pillar = result['四柱']['月柱']['干支']
day_pillar = result['四柱']['日柱']['干支']
hour_pillar = result['四柱']['时柱']['干支']

# 读取五行统计
wuxing_count = result['五行统计']

# 读取刑冲合害
xingchong = result['刑冲合害']

# 读取神煞
shensha = result['四柱神煞']
```

#### 示例4：指定流年年份
```python
from bazi_calculator import calculate_bazi, print_bazi

# 计算2025年流年
result_2025 = calculate_bazi(
    birth_time_str="1990-05-15 08:30",
    location="成都",
    gender="男",
    name="张三",
    liunian="2025"  # 指定流年
)
print_bazi(result_2025)

# 计算2030年流年
result_2030 = calculate_bazi(
    birth_time_str="1990-05-15 08:30",
    location="成都",
    gender="男",
    name="张三",
    liunian="2030"
)
print_bazi(result_2030)

# 不指定流年（使用当前年份）
result_current = calculate_bazi(
    birth_time_str="1990-05-15 08:30",
    location="成都",
    gender="男",
    name="张三"
    # liunian 不传或传 None，自动使用当前年份
)
print_bazi(result_current)
```

## 支持的城市

程序内置了主要城市数据库，支持以下城市：
- 北京、上海、广州、深圳、成都、重庆、西安、武汉、南京、杭州、天津、苏州、长沙、郑州、东莞、青岛、沈阳、宁波、昆明、大连、哈尔滨、长春、厦门、济南、合肥、福州、石家庄、南宁、贵阳、南昌、太原、乌鲁木齐、兰州、呼和浩特、海口、银川、西宁、拉萨等

## 时间格式支持

支持以下时间格式：
- `YYYY-MM-DD HH:MM:ss`（精确到秒）
- `YYYY-MM-DD HH:MM`（精确到分钟，推荐）
- `YYYY/MM/DD HH:MM`（斜杠分隔）

## 出生地格式支持

支持以下格式：
1. 城市名：`成都`、`北京市`、`上海市`
2. 经纬度：`104.07,30.67` 或 `104.07, 30.67`（经度在前，纬度在后）

## 注意事项

1. **真太阳时修正**：程序会根据出生地经度自动计算真太阳时，无需手动调整
2. **节气精确计算**：使用 VSOP87 简化算法，精度 ±1 分钟
3. **空亡算法**：按六十甲子旬空原则，年月日时各柱独立计算
4. **流年干支确定**：以立春为界确定流年干支，立春前使用上一年干支
5. **输出文件编码**：所有输出文件均使用 UTF-8 编码
6. **目录作为输出路径**：如果输出路径是目录，会自动生成带时间戳的文件名

## 错误处理

### 常见错误及解决方法

1. **时间格式错误**
   ```
   错误：无法解析时间格式：1990/5/15 8:30，请使用 YYYY-MM-DD HH:MM
   解决：使用标准格式 1990-05-15 08:30
   ```

2. **城市未找到**
   ```
   警告：[未找到城市] '某城市'，使用北京坐标（116.41°E, 39.90°N）
   解决：使用其他城市名或直接输入经纬度
   ```

3. **权限错误**
   ```
   错误：Permission denied: 'D:\某目录'
   解决：确保输出目录有写入权限，或提供完整的文件路径
   ```

## 交互模式

除了命令行参数，程序还支持交互模式。直接运行程序而不提供 `--time` 参数即可进入交互模式：

```bash
python bazi_calculator.py
```

交互模式会依次提示输入：
1. 出生时间（格式：YYYY-MM-DD HH:MM）
2. 出生地（城市名，如：成都）
3. 性别（男/女）
4. 姓名（留空则不显示）
5. 流年年份（格式：YYYY，留空则使用当前年份）
6. 输出文件路径（留空则输出到控制台）
